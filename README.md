## Install my system scripts  

```shell
 sudo bash -c "$(curl -q -LSsf "https://github.com/systemmgr/installer/raw/main/install.sh")"
 sudo systemmgr --config && sudo systemmgr install scripts  
```

## Get source files  

```shell
dockermgr download src php7
```

```shell
git clone "https://github.com/casjaysdevdocker/php7" "$HOME/Projects/github/casjaysdevdocker/php7"
```

```shell
cd "$HOME/Projects/github/casjaysdevdocker/php7"
buildx 
```
